QTool 是一个开源,免费的 Xposed 模块,在使用之前,你应该进行百度了解Xposed是啥,如何使用,最新的构建版本可以到Action中进行下载

## 说明

- 支持Android >= 8.1
- 支持QQ 版本为8.8.35以上,由于作者精力有限无法保证更新适配的时效性和可用性,请确定自己的需求再使用
- 本模块功能不是很全面,建议同时配合其他模块使用

## 功能列表
- 自行安装模块查看
